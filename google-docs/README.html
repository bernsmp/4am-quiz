🆘 Need Help?

Common Issues:

Q: Schema validation shows errors
A: Check for missing commas, brackets, or incorrect date formats. Dates must be YYYY-MM-DD.

Q: Can't upload robots.txt
A: You may need FTP access. Contact your hosting provider or use cPanel File Manager.

Q: FAQ accordion doesn't work
A: Make sure you copied the entire faq-schema.html file including the JavaScript at the bottom.

Q: Google Docs import looks broken
A: Use Method 1 (Direct Upload) from the import instructions - it preserves formatting best.
