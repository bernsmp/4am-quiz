45 minutes

üÜò Troubleshooting

Schema Validation Errors

Problem: "Missing required field"

Solution: Double-check you copied the complete schema. Missing commas or brackets cause this.

Problem: "Invalid date format"

Solution: Dates must be in format: YYYY-MM-DD or YYYY-MM-DDT00:00:00-06:00

Schema Not Showing in Test

Problem: Rich Results Test shows no schema

Solution: Check that you added schema to the <head> section, not the body. View page source and search for "application/ld+json"

FAQ Accordion Not Working

Problem: FAQ items don't expand/collapse

Solution: Make sure you copied the ENTIRE faq-schema.html file including the JavaScript at the bottom

Can't Upload robots.txt

Problem: File upload blocked or overwritten

Solution: Some hosts require FTP access. Contact your hosting provider or use cPanel File Manager.
